# 铁脊柱 · Iron Spine

> 开源被动式外骨骼——弯腰助力、模块化、零电子元件、日常材料制造
>
> "最好的技术不是最贵的，是能用废品站材料造出来的。"

---

## 项目简介

一套**完全开源**的被动式外骨骼设计，服务于建筑工人、搬运工、仓储装卸工。利用弹簧/弹性杆的物理储能原理——弯腰时储存势能，起身时释放——减轻腰椎负担。

- **目标成本**：¥300-500/套（对比：美国 SuitX 被动外骨骼 $3,000+）
- **核心材料**：玻纤板、自行车内胎弹力绳、PLA 3D 打印件、尼龙绑带
- **设计原则**：零电子元件、5 分钟穿戴、模块化（背撑/膝撑/肩撑可拆）、坏了好修

## 阅读路线图 · How to Read This Code

### 第一层：理解硬件设计（15 分钟）

打开 `docs/`，按以下顺序读：

1. **`docs/material-specs.md`** — 先理解材料。每一行是一个材料条目：玻纤板（哪里买、替代品、裁切建议）、弹力绳（规格、如何测试弹性系数）、PLA 打印件（STL 文件位置、填充率建议）。你读完之后就知道整套外骨骼用了什么东西、为什么选这些。

2. **`docs/build-guide-zh.md`** — 组装步骤。分三章：背撑模块 → 膝撑模块 → 肩撑模块。每一章的结构是：材料清单 → 工具清单 → 步骤（配图）。读完就知道怎么从零做出一套。

3. **`designs/`** — 三个 SVG 设计图。每张图标注了关键尺寸和力学节点。背撑是核心模块（承受弯腰力矩），膝撑是辅助（分散膝盖压力），肩撑是可选（搬运时用）。

### 第二层：理解配置器（10 分钟）

打开 `index.html`，这是一个 **Web 配置器**，让使用者输入自己的身体参数，自动生成定制化的设计参数和物料清单。

1. **`js/configurator.js`** — 核心算法。`Configurator` 类：
   - `calculate(userParams)` — 输入身高/体重/腰围/工作类型 → 输出杆长、弹力绳规格、绑带长度
   - 算法基于人体工学：腰椎 L5-S1 节段承受最大力矩，弹力绳预张力 = 躯干重量的 60%
   - `generateSTLPreview(params)` — 生成 3D 打印件的关键参数（后续可对接 STL 生成 API）

2. **`js/bom-generator.js`** — `BOMGenerator` 类：
   - `generate(params)` — 根据设计参数自动生成购物清单
   - 每条材料包含：名称、规格、数量、淘宝搜索关键词、替代品、预估价格
   - `exportMarkdown(bom)` — 导出可打印的 Markdown 清单

3. **`js/design-viewer.js`** — `DesignViewer` 类：
   - 在 Canvas 上渲染 2D 设计图
   - 支持缩放和标注切换
   - `highlightModule(moduleName)` — 高亮特定模块

4. **`js/app.js`** — `IronSpine` 类：
   - 管理配置器四个步骤：输入参数 → 设计预览 → 物料清单 → 组装指南
   - 串联 configurator → bom-generator → design-viewer

### 第三层：理解社区功能设计（5 分钟）

社区功能是铁脊柱的重要组成部分（见 `index.html` 底部的社区面板）：
- **STL 生成器** — 身体参数 → 定制 3D 打印件 → 下载 STL
- **物料清单自动生成** — 输入当地可用的材料 → 自动调整 BOM
- **多语种组装视频** — 设计图标注 → AI 配音讲解
- **反馈社区** — 使用者上传改进方案和实际使用效果

## 运行方式

### Web 配置器
```bash
python -m http.server 8000
# 浏览器打开：http://localhost:8000/iron-spine/
```

### 使用流程
1. 输入身高、体重、腰围、主要工作姿势
2. 系统自动计算设计参数
3. 查看 2D 设计预览
4. 生成物料清单（含淘宝关键词和预估价格）
5. 对照组装指南开始制作

## 技术选型

| 层 | 技术 | 原因 |
|----|------|------|
| 设计图 | SVG（矢量） | 任意缩放不失真，可打印 |
| 力学计算 | JavaScript（客户端） | 零服务器成本，离线可用 |
| STL 生成 | OpenSCAD / OpenJSCAD | 参数化 3D 建模，纯代码 |
| 3D 预览 | Three.js（后续） | WebGL 直接渲染 STL |
| 社区 | GitHub Issues + Discussions | 零成本，可持续 |

## 设计哲学

- **废品站材料原则** — 性能可以降 15%，成本必须降 95%
- **看得懂的图纸** — 让你妈也能看懂。不用专业术语，用图示
- **坏了好修** — 模块化即修复性。哪个零件坏了换哪个，不需要送回工厂
- **不做拯救者** — 这是工具，不是慈善。使用者是最懂自己身体的人
