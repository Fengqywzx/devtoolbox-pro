## 第一章：背撑模块（核心）

### 你需要准备的材料
- [ ] 玻纤板 1块（尺寸看配置器输出）
- [ ] 弹力绳（规格和长度看配置器输出）
- [ ] 尼龙织带（5cm宽）
- [ ] 3D打印件：腰带扣座 ×2、弹力绳锚点 ×4、杆端连接件 ×2
- [ ] M5不锈钢螺栓螺母 ×8套
- [ ] M3不锈钢螺栓螺母 ×4套
- [ ] 尼龙垫片 ×16个
- [ ] EVA泡棉垫（5×30cm左右）
- [ ] 魔术贴

### 工具
- [ ] 电钻 + 3mm/5mm钻头（或手钻+锉刀）
- [ ] 剪刀 / 美工刀
- [ ] 卷尺
- [ ] 打火机
- [ ] 内六角扳手一套
- [ ] 砂纸（200目）

### 步骤

**1. 裁玻纤板**
用卷尺在玻纤板上量好长度，画线。用钢锯沿线锯断。锯完后用砂纸打磨边缘（非常重要——玻纤毛刺扎手）。戴口罩和护目镜。

**2. 打孔**
在玻纤板两端各打2个M5孔。孔的位置：距板端3cm，两孔间距4cm。
打好后检查：螺栓能自由穿过但不太松。

**3. 装锚点**
将弹力绳锚点用M5螺栓固定在板两端的孔上。螺栓穿过锚点→玻纤板→垫片→螺母。不要拧太紧，等全部装好再统一拧紧。

**4. 装腰带扣座**
腰带扣座用M3螺栓固定在玻纤板中部（大约在板的正中间，上下误差2cm没关系）。

**5. 穿弹力绳**
将弹力绳穿过上下两个锚点的孔，调整长度使弹力绳刚好有一点张力（不完全松垮）。两端打结或用绳夹固定。

**6. 穿腰带**
尼龙带穿过腰带扣座的槽口。调节带长：比你腰围长30cm（给冬天穿厚衣留余量）。

**7. 贴泡棉**
将EVA泡棉贴在腰带内侧（接触腰的部分）。用剪刀修剪成贴合腰部的形状。用魔术贴或双面胶固定。

### 调试方法

穿上，系紧腰带。慢慢弯腰：
- **太紧**（弯腰困难）→ 放长弹力绳1-2cm
- **太松**（感觉不到助力）→ 缩短弹力绳1-2cm  
- **刚好**：弯腰45°时弹力绳开始明显拉伸，起身时感到轻柔的推力

调试可能需要反复3-5次，耐心调整。

### 常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| 腰带往下滑 | 腰带不够紧或臀部没托住 | 收紧腰带，或在腰带下加一条细带勾住裤腰 |
| 弹力绳磨衣服 | 锚点位置偏内 | 在锚点外加一个垫片，把弹力绳往外移 |
| 弯腰时咯吱响 | 螺栓松动 | 检查并拧紧所有螺栓 |
| 3D打印件开裂 | 材料太脆或填充率不够 | 更换为PLA+，填充率提高到60% |

## 第二章：膝撑模块（可选，蹲跪工种加装）

### 额外材料
- [ ] 弹力绳（膝撑用，约60cm）
- [ ] 尼龙带（5cm宽 × 80cm）×2
- [ ] 3D打印件：膝部铰链座 ×2
- [ ] EVA泡棉垫（膝窝用）

### 步骤

**1. 大腿绑带** — 尼龙带绕大腿固定（膝盖上方10cm），铰链座上端连接大腿绑带。
**2. 小腿绑带** — 尼龙带绕小腿固定（膝盖下方10cm），铰链座下端连接小腿绑带。
**3. 装弹力绳** — 弹力绳连接铰链座的弹簧挂点，与背撑弹力绳方向一致。
**4. 同步调试** — 先单独调膝撑弹力绳，再和背撑一起穿，做几个深蹲测试。

### 调试要点
- 膝撑弹力绳应该比背撑的张力小（约为60%），否则膝盖不舒服
- 铰链座必须正对膝关节旋转轴，偏了会磨损膝盖

## 第三章：肩撑模块（可选，搬运重物工种加装）

### 额外材料
- [ ] 弹力绳（肩撑用，约80cm）×2条
- [ ] 3D打印件：肩部支架 ×2
- [ ] 窄尼龙带（2.5cm宽 × 60cm）×2条（肩部固定）

### 步骤

**1. 固定肩部支架** — 窄尼龙带穿过肩部支架的槽，固定在肩胛骨上方（不是你想象的肩膀顶部，而是肩胛骨上方）。
**2. 连接弹力绳** — 弹力绳从肩部支架后方的锚点连接到背撑的腰带扣座。
**3. 调试** — 双臂自然下垂时弹力绳轻微张力；搬抬重物时弹力绳拉伸辅助手臂。

### 调试要点
- 肩部支架不能压在锁骨上——往前调整1-2cm
- 弹力绳太粗会影响手臂活动范围，选比背撑小一号

## 附录：大白话术语表

| 术语 | 大白话 |
|------|--------|
| 锚点 | 弹力绳固定在上面的那个小零件，作用类似"挂钩" |
| 预张力 | 弹力绳在你还没弯腰时就已经拉着的力，太紧=一直拽着你，太松=没用 |
| 力矩 | 可以理解为"腰部承受的压力大小"，弯腰拿东西时最大 |
| 填充率 | 3D打印件的"实心程度"，30%=里面70%是空的但省钱，60%=更结实但费材料 |
| FR4 | 玻纤板的正式名字，电路板用的那种玻璃纤维板，很结实 |
| PLA | 3D打印机最常用的塑料，玉米做的环保材料，便宜但不太耐热 |
| PLA+ | PLA的升级版，更韧一点，适合做受力件 |

## 安全警告

1. **前3天每天只穿2小时** — 让身体适应
2. **弹力绳断裂** — 如发现绳子起毛、开裂，立即更换
3. **不要穿着睡觉**
4. **这不是医疗器械** — 如果你已经确诊腰椎间盘突出，请先咨询医生
5. **PLA打印件不耐热** — 不要放在夏天暴晒的车里
