# 铁脊柱 — 材料规格与替代方案

## 核心材料

### 1. 玻纤板（FR4 玻璃纤维增强环氧树脂板）

| 项目 | 规格 |
|------|------|
| 用途 | 背撑主支撑杆 |
| 推荐规格 | 50cm × 10cm × 3mm |
| 淘宝关键词 | "FR4 玻纤板 3mm" |
| 价格 | ¥30-40/块 |
| 加工方式 | 钢锯裁切 + 电钻打孔 |
| 安全提示 | 裁切时必须戴口罩和护目镜，玻纤粉尘刺激皮肤 |

**替代方案（按推荐度排序）：**
1. 竹片（3层交叉纹粘合）— 免费（农村）/¥5-10，性能降15%
2. 6061铝合金条 — ¥15-25，性能降5%，但加工更难（需角磨机）
3. 硬木板（橡木/榉木）— ¥10-20，性能降20%，耐久性差
4. PVC管（直径25mm）— ¥5-10，性能降30%，不推荐长期使用

---

### 2. 弹力绳/弹力带

| 项目 | 规格 |
|------|------|
| 用途 | 弯腰储能 + 起身释放 |
| 推荐规格 | 8mm直径，天然乳胶芯，尼龙编织外皮 |
| 淘宝关键词 | "弹力绳 8mm 高弹力" |
| 价格 | ¥15-20/米 |
| 使用寿命 | 6-12个月（高频使用）/ 12-18个月（偶尔使用） |

**替代方案：**
1. 自行车内胎条（纵向剪裁）— 废品站免费或¥2-5/条，性能降20%，寿命3-6月
2. 健身弹力带裁剪 — ¥8-15/米，性能相当
3. 橡胶管（五金店）— ¥3-5/米，性能降30%，气味大

**弹力测试方法：** 挂5kg重物，弹力绳伸长量应为原长的50%-80%。差距过大则不适合。

---

### 3. PLA / PLA+ 3D打印件

| 项目 | PLA | PLA+ |
|------|-----|------|
| 用途 | 非承重连接件 | 承重连接件（锚点、扣座） |
| 推荐填充率 | 30% | 50-60% |
| 价格 | ¥25-30/卷(1kg) | ¥35-45/卷(1kg) |
| 耐温 | 60°C软化 | 65-70°C软化 |
| 淘宝关键词 | "PLA 3D打印耗材 1kg" | "PLA+ 高强度 1kg" |

**无打印机替代方案：**
- 淘宝代打服务：¥40-60/件（搜索"3D打印代打 PLA"）
- ABS板热弯 + 手工打磨 — 技术要求高，不推荐新手
- 硬木雕刻 — 只有传统手艺人才推荐

## 废品站采购指南

以下材料可以在废品站/修理摊免费或极低价获取：

| 材料 | 来源 | 检查要点 |
|------|------|----------|
| 自行车内胎 | 修车摊（废胎） | 无龟裂、无大面积补丁 |
| 汽车安全带 | 报废车场 | 无磨损、锁止机构完好 |
| PVC管 | 建筑工地废料 | 无裂痕、壁厚均匀 |
| 泡棉/海绵 | 旧沙发/床垫 | 无异味、弹性好 |
| 木板条 | 装修废料 | 无虫蛀、纹理直 |
