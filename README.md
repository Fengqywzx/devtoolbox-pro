# DevToolbox Pro

程序员效率工具箱 — 8 合 1 开发辅助工具，单个 HTML 文件，打开即用。

## 工具列表

| 工具 | 功能 |
|------|------|
| JSON 格式化 | 格式化、压缩、验证 JSON |
| Base64 编解码 | 文本 ↔ Base64 互转 |
| URL 编解码 | encodeURIComponent / decodeURIComponent |
| 时间戳转换 | Unix 时间戳 ↔ 可读日期 |
| 颜色工具 | HEX ↔ RGB ↔ HSL 互转 + 色板 |
| 正则测试 | 编写正则，实时测试匹配 |
| 文本对比 | 逐行比较差异 |
| UUID 生成 | UUID v4 批量生成 |

## 特点

- 暗色主题，GitHub 风格 UI
- 无需安装，浏览器打开即用
- 无需联网，数据不上传第三方
- 单文件，方便携带和分享

## 使用

```
浏览器打开 devtoolbox-pro.html
```
